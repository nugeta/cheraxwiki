> **Note:** This site is not official, these tutorials are written and borrowed by gustigost from the Cherax Discord Server.
In the case that you would like to help contribute in any way to the site, like for example adding your own tutorials or donating to help upkeep the site, feel free to reach out to me.

## Table of Contents

- **Purchasing Cherax**: Explains how to buy from resellers, how to save money and similar when purchasing Cherax.
- **Getting Started**: A generic rundown of the menu, how to set it up, and how to prevent errors when running it.
- **Getting Full Access to Channels**: Explains how to get unrestricted chat access in the Cherax discord and also get all the relevant channels.
- **Outfits**: Explains how to download outfits whilst also showing how to load them into the game and keep them.
- **Modded Vehicles**: Explains how to download modded vehicles whilst also showing how to load them into the game and keep them.
- **Menu Themes**: Explains how to download themes whilst also showing how to load them into the game and keep them.
- **DLCs**: Explains how to use DLC vehicles and peds in game to have a little fun.
- **Luas**: An overview of Lua Scripts like Silent Night, Codax, Bunnyscript, Jinxscript and many others. Also shows how to run and configure them.
- **Advertising**: How to advertise a session to get players in, fixing some common issues and using port forwarding or NAT type spoofing for a better advertising experience.
- **Recoveries**: An overview of most, if not all the recovery features, as well as how to stay safe whilst doing them.
- **Prime**: Explains everything you need to know about Cherax Prime.
- **Donations**: Shows information on how you can help fund the upkeeping of the wiki.
- **Contributions**: Shows information on how you can add or request content to be added/removed from the wiki.
