In the case that you would like to help contribute in any way to the site, like for example adding your own tutorials or donating to help upkeep the site, feel free to reach out to me.

## Sections in this wiki (may update sooner or later)

- **Getting Started**: A generic rundown of the menu, how to set it up, and how to prevent errors when running it.
- **Outfits, Vehicles, Themes, etc**: Explains how to download outfits, as well as vehicles, themes and etc whilst also showing how to load them into the game and keep them.
- **Luas**: An overview of Lua Scripts like Silent Night, Codax, Bunnyscript, Jinxscript and many others. Also shows how to run and configurate them.
- **Advertising**: How to advertise a session to get players in, fixing some common issues and using port forwarding or NAT type spoofing for a better advertising experience.
- **Recoveries**: An overview of most, if not all the recovery features, as well as how to stay safe whilst doing them.
